# Keyword Strategy

## Primary clusters
- ทีวีและเครื่องใช้ไฟฟ้า
- ไอทีและอุปกรณ์
- บ้านและครัว
- มอเตอร์ไซค์และยานยนต์

## Workflow
1. Clean keyword
2. Normalize keyword
3. Classify category
4. Classify intent
5. Score priority
6. Group into cluster
7. Assign article type
8. Map products

## Initial target
- Top 30 keywords
- 5 candidate products per keyword
- 3 selected products per article
