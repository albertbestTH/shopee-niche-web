# Roadmap

## Phase 1 — Foundation
- Initialize repository
- Define project structure
- Define data model
- Prepare keyword source data

## Phase 2 — Keyword Intelligence
- Clean keywords
- Cluster keywords
- Classify intent
- Score priority
- Lock Top 30 keywords

## Phase 3 — Product Research
- Research 5 products per keyword
- Select 3 products per article
- Store product metadata
- Prepare affiliate URLs

## Phase 4 — Website MVP
- Home page
- Category pages
- SEO article pages
- Product comparison sections
- Internal linking

## Phase 5 — SEO Automation
- Metadata
- JSON-LD
- Sitemap
- Robots.txt
- Breadcrumbs

## Phase 6 — Analytics
- Click tracking
- Affiliate conversion tracking
- Search Console integration
- Content performance dashboard
